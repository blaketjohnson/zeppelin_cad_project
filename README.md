A **Python-based design environment** for a conceptual **Zeppelin airship**.  
Implements parametric modeling of engines, fins, envelope, and gondola, with both **Python modules** and a packaged **GUI executable**.  
Includes final report, presentation slides, and video demo as part of a complete engineering design portfolio.

---

## ✨ Highlights for Recruiters
- Modular Python codebase with parametric Zeppelin components.  
- Configurable via `config.py` with geometry & performance parameters.  
- GUI executable (`Zeppelin.CAD.exe`) available via GitHub Releases.  
- Documentation includes **final report, slides, user guide, and video**.  
- Demonstrates **engineering design, software packaging, and technical presentation skills**.

## 🚀 Quick Start

Clone and install dependencies:
```bash
git clone https://github.com/blaketjohnson/zeppelin_cad_project.git
cd zeppelin_cad_project
pip install -r requirements.txt
```

Run the Python entry point:
```bash
python cad/main.py
```

Executable GUI is available via [Releases](../../releases).
